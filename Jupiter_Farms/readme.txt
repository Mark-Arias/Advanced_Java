Jupiter Farms is a project designed for the Advanced Java course at Santa Monica College.
This project is designed to exemplify and apply the core principles of software engineering.
By this I mean that this project utilized multiple design phases from general discuession of 
project purpose, to UML drafting, to code architercure design, to coding implementation,
debugging, testing, and project deployment.

Project Purose:
Essentially this is a small arcade game based on source code freely available on the internet
